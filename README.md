# Facundo Leis Pou — Currículo (GitHub Pages)

Static résumé site and PDF for job applications.

- **Repository:** [github.com/ezeleis/curriculo](https://github.com/ezeleis/curriculo)
- **Published site:** [ezeleis.github.io/curriculo/](https://ezeleis.github.io/curriculo/)

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
npx --yes serve .
```

## GitHub Pages

This repo is configured for **GitHub Pages** (typically **Deploy from a branch** → `main` → `/ (root)`). After you push, the live URL above updates within a few minutes.

## PDF for applications

- **Latest export:** `Facundo_Leis_Pou_Resume_export.pdf` — generated by the script below. The site **Download PDF** button saves it as `Facundo_Leis_Pou_Resume_FullStack.pdf` for applications.
- If you prefer a single canonical filename in the repo, close any PDF viewer that has the old file open, then replace `Facundo_Leis_Pou_Resume_FullStack.pdf` with the export (or rerun the script so it can overwrite).
- **Regenerate:** open `index.html`, use **Print / Save as PDF**, or from this folder run:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\export-pdf.ps1
```

(requires Microsoft Edge).

## Deploy / push updates

From this folder:

```bash
git remote add origin https://github.com/ezeleis/curriculo.git
git push -u origin main
```

If `origin` already exists, use:

```bash
git remote set-url origin https://github.com/ezeleis/curriculo.git
git push -u origin main
```
